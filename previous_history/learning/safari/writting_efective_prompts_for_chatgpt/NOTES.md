# Writing Effective Prompts for ChatGPT


## Prompts


### Prompt 01
```text
Teach me something new
```

### Prompt 02
```text
Explain that again in one sentence that a 5 year old would understand
```

### Prompt 03 (Explain like I'm 5).  ELI5 stands for "explain like I'm 5." 
```text
ELI5 in 1 sentence
```
